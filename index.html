APIPmaValidateToken

Endpoint de validación de tokens de sesión PMA. Recibe un token UUID y retorna los datos del usuario y empresa asociados, o un código de error en caso de que el token no sea válido.

04
Respuestas del servidor
HTTP 200 en todos los casos — revisar campo Success
200 OK Token válido — Autenticación exitosa SUCCESS ▾

El token existe, es válido y no ha sido utilizado. Se retornan los datos del usuario y empresa asociados al token.

JSON · Response 
{
  "SDT_PmaResponse": {
    "Success":      true,
    "UserEmail":    "edisonmauricio@expoflores.com",
    "EmpresaRuc":   "1790986055001",
    "UserProfile":  "AUDITOR_FE",
    "ErrorCode":    "",
    "ErrorMessage": ""
  }
}
CampoValorDescripción
SuccesstrueValidación exitosa
UserEmailEmail del usuarioCorreo registrado en PMA
EmpresaRucRUC empresaRUC de la empresa asociada
UserProfilePerfilEj: AUDITOR_FE, ADMIN…
ErrorCodevacíoSin error
ErrorMessagevacíoSin mensaje de error
200 OK Token no encontrado TOKEN_NOT_FOUND ▾

El token enviado no existe en el sistema PMA o su formato es incorrecto. Verificar que el UUID sea exacto y esté completo.

JSON · Response 
{
  "SDT_PmaResponse": {
    "Success":      false,
    "UserEmail":    "",
    "EmpresaRuc":   "",
    "UserProfile":  "",
    "ErrorCode":    "TOKEN_NOT_FOUND",
    "ErrorMessage": "El token proporcionado no es válido o no existe."
  }
}
200 OK Token ya utilizado TOKEN_ALREADY_USED ▾

El token ya fue procesado en una sesión anterior. Los tokens PMA son de un solo uso — se debe solicitar uno nuevo al sistema.

JSON · Response 
{
  "SDT_PmaResponse": {
    "Success":      false,
    "UserEmail":    "",
    "EmpresaRuc":   "",
    "UserProfile":  "",
    "ErrorCode":    "TOKEN_ALREADY_USED",
    "ErrorMessage": "El token ya ha sido procesado anteriormente."
  }
}
200 OK Token expirado TOKEN_EXPIRED ▾

El token existe en el sistema pero su tiempo de validez ha vencido. Se debe solicitar un nuevo token al sistema PMA para reiniciar el flujo.

JSON · Response 
{
  "SDT_PmaResponse": {
    "Success":      false,
    "UserEmail":    "",
    "EmpresaRuc":   "",
    "UserProfile":  "",
    "ErrorCode":    "TOKEN_EXPIRED",
    "ErrorMessage": "El tiempo de validez del token ha expirado."
  }
}
